# AI-RPCT Public Beta Capabilities

AI-RPCT is a public beta terminal for AI infrastructure intelligence.

Current live capabilities:

- Vast.ai live provider feed
- RunPod live provider feed
- GPU price index
- GPU price history
- GPU rankings
- Provider reliability
- Provider comparison
- Frontier GPU index
- GPU watchlist intelligence
- Intelligence signal score
- Daily terminal brief
- Weekly infrastructure report
- Public web terminal
- Railway deployment

Current limitations:

- RunPod pricing requires deeper enrichment
- Additional providers needed
- Longer historical data window needed
- Forecast accuracy needs more time-series data
