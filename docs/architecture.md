# AI-RPCT Architecture

AI-RPCT is an AI infrastructure intelligence platform for collecting, normalizing, forecasting, validating, and governing GPU cloud market signals.

## Core Layers

1. Live provider ingestion
2. Historical provider archive
3. Unified historical dataset
4. Feature Store
5. Forecast engines
6. Outcome tracking
7. True outcome labels
8. ML retraining gate
9. Promotion guard
10. Readiness dashboard

## Current Platform Stage

AI-RPCT is currently in pilot stage.

Customer previews are allowed, but paid production forecasting claims remain blocked until true future outcome labels mature and models are validated on them.
