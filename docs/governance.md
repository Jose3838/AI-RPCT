# Governance

AI-RPCT uses governance gates to prevent premature production claims.

## Key Gates

- Leakage Audit
- Training Dataset v3 Readiness
- Auto Retraining Manager
- Production Promotion Guard
- Customer Forecast Readiness
- AI-RPCT Readiness Dashboard

## Current Rules

Customer demo is allowed.

Paid production forecasting is blocked until:

- true future labels are available
- retraining is allowed
- leakage checks pass
- benchmark evidence supports promotion
- production promotion guard allows release
