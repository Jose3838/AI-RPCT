# CUDA Timeline v1

## Purpose

This registry introduces a historical CUDA software timeline for AI-RPCT.

It provides source-backed CUDA release references for future historical enrichment, software-stack analysis, and feature-store development.

## Dataset

Primary dataset: data/cuda_timeline.csv

Warehouse copy: warehouse/historical/cuda/cuda_timeline.csv

## Scope

Initial coverage includes selected CUDA release documentation references:

- CUDA 9.0
- CUDA 10.0
- CUDA 12.0 Update 1
- CUDA 12.3
- CUDA 13.0 Update 1

## Governance

Rules:

- source-backed documentation references only
- no inferred support matrices
- no copied benchmark claims
- no pricing
- no forecast accuracy claims
- no synthetic labels

## Known Gaps

Future versions should expand:

- CUDA 11.x releases
- CUDA 12.x release sequence
- minimum driver versions only when source-backed
- architecture support and deprecation fields only when source-backed
