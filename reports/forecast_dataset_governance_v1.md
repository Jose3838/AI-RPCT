# Forecast Dataset Governance v1

## Purpose

This report documents the governance status of Forecast Dataset v1.

## Dataset Status

Forecast Dataset v1 is feature-only.

It does not contain:

- true future outcome labels
- verified prediction targets
- model training labels
- performance benchmarks
- accuracy metrics

## Governance Decision

ML training is not allowed from Forecast Dataset v1.

Production promotion is not allowed from Forecast Dataset v1.

Paid production use is not allowed from Forecast Dataset v1.

## Allowed Usage

Forecast Dataset v1 may be used for:

- schema validation
- feature pipeline validation
- dashboard prototyping
- explainability pipeline prototyping
- non-production analytics

## Blocked Usage

Forecast Dataset v1 must not be used for:

- ML training
- accuracy claims
- benchmark claims
- paid production forecasting
- automated promotion decisions
