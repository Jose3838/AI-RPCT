# AI-RPCT Core Capabilities

## Vision

AI-RPCT is not a GPU tracker.

AI-RPCT is a Decision Intelligence Platform for AI Infrastructure.

------------------------------------------------------------

## Capability 1

Market Collection

Purpose

Continuously collect infrastructure market information.

Sources

- GPU Providers
- Cloud Providers
- Official Releases
- Historical Registries
- Market Signals

------------------------------------------------------------

## Capability 2

Infrastructure Intelligence

Purpose

Understand what is happening.

Examples

- Provider Rankings
- Scarcity
- Supply
- Demand
- Capacity
- Pricing

------------------------------------------------------------

## Capability 3

Forecast Intelligence

Purpose

Predict what is likely to happen.

Examples

- Capacity Forecast
- GPU Forecast
- Supply Forecast
- Risk Forecast
- Provider Forecast

------------------------------------------------------------

## Capability 4

Decision Intelligence

Purpose

Recommend actions.

Examples

- Buy now
- Wait
- Reserve Capacity
- Switch Provider
- Diversify Vendors

------------------------------------------------------------

## Capability 5

Executive Intelligence

Purpose

Explain decisions.

Every recommendation must answer

Why?

Confidence?

Evidence?

Risk?

Alternatives?

------------------------------------------------------------

## Capability 6

Data Moat

Purpose

Create a competitive advantage.

Built from

- Historical data
- Forecast history
- Provider history
- Decision history
- Snapshot history
- Registry history

------------------------------------------------------------

## Capability 7

Enterprise Platform

Purpose

Deliver intelligence.

Channels

- API
- Web Console
- Reports
- Customer Dashboard
- Executive Dashboard

------------------------------------------------------------

North Star

Help organizations make better AI infrastructure decisions.
