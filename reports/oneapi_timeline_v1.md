# oneAPI Timeline v1

## Purpose

This registry introduces a historical oneAPI software timeline for AI-RPCT.

It provides source-backed oneAPI release references for future software-stack analysis, feature-store development, and historical enrichment.

## Dataset

Primary dataset: data/oneapi_timeline.csv

Warehouse copy: warehouse/historical/oneapi/oneapi_timeline.csv

## Scope

Initial coverage includes selected oneAPI release references:

- oneAPI 2021.1
- oneAPI 2022.1
- oneAPI 2023.2
- oneAPI 2024.2

## Governance

Rules:

- source-backed documentation references only
- no inferred support matrices
- no copied benchmark claims
- no pricing
- no forecast accuracy claims
- no synthetic labels

## Known Gaps

Future versions should expand:

- complete oneAPI release sequence
- component-level releases only when source-backed
- deprecation and lifecycle fields only when source-backed
