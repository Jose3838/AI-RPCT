# ROCm Timeline v1

## Purpose

This registry introduces a historical ROCm software timeline for AI-RPCT.

It provides source-backed ROCm release references for future software-stack analysis, feature-store development, and historical enrichment.

## Dataset

Primary dataset: data/rocm_timeline.csv

Warehouse copy: warehouse/historical/rocm/rocm_timeline.csv

## Scope

Initial coverage includes selected ROCm release references:

- ROCm 5.0
- ROCm 6.0
- ROCm 6.1
- ROCm 7.0

## Governance

Rules:

- source-backed documentation references only
- no inferred support matrices
- no copied benchmark claims
- no pricing
- no forecast accuracy claims
- no synthetic labels

## Known Gaps

Future versions should expand:

- ROCm 5.x release sequence
- ROCm 6.x release sequence
- supported GPU families only when source-backed
- OS and driver compatibility only when source-backed
