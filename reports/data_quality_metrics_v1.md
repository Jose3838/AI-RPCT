# Data Quality Metrics v1

## Purpose

The Data Quality Metrics registry summarizes basic quality metrics for generated datasets.

## Metrics

- row count
- column count
- generation status

## Datasets

- feature_store.csv
- forecast_dataset.csv
- forecast_engine_v1_output.csv
- forecast_explanations.csv

## Governance

This registry reports metadata only.

No forecasts, predictions, or ML performance metrics are included.
